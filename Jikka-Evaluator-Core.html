Jikka.Evaluator.Core

Description

Core evaluates exprs to values. Also this recognizes users' inputs at once.

The implementation assumes that all variable names don't conflict even when their scopes are distinct. Also this assumes that exprs allow the eager evaluation. Please use MakeEager if needed.
